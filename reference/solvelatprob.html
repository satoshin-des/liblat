liblat　関数レファレンスページ

その他格子問題の求解に関する関数

宣言 std::vector<T> babaiNearPlane(const std::vector<double> target);
概略 Babaiの最近平面法
引数
std::vector<double> ： 目標ベクトル
戻り値 目標ベクトルに近い格子ベクトル
解説 Babaiの最近平面法を用いて，目標ベクトルtargetの近似最近ベクトルを求めます．

サンプル

#include <iostream>
#include "lattice.h"

int main()
{
    // 3次正方行列を基底行列に持つ格子
    Lattice<int> lat(3, 3);

    // 目標ベクトルの設定
    std::vector<double> v = {19, 78, -9};

    // 近似最近ベクトルの表示
    print(lat.babaiNearPlane(v));

    return 0;
}
